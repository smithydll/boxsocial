Simple permissions replaces all detailed permissions with the simple permissions below once you hit save. This may enable extra permissions to people who could not access content or a function beforehand.

If a permission block is empty, permissions are inherited from the next level up. It is recommended to set view permissions for each content item.

Detailed permissions are for people who understand an access control lists base permissions system. Permissions can be set granularly for each function of each content item.
